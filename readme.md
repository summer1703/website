
### 前台项目

本仓库平时学习各框架所用,接口依赖[nodeback](https://github.com/ExcaliburZhou/nodeback)

* [require-project]

    require+jquery+gulp

* [react-project]

    react+redux+webpack

安装步骤

   * git clone git@github.com:ExcaliburZhou/website.git
   * cd react-project or cd require-project
   * npm i -g webpack-dev-server or npm i -g gulp
   * npm install
   * npm start
